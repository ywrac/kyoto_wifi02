kyoto_wifi02
============

Requirements
----------------
* python
* requests

How to setup
---------------
In kyoto_wifi02 root directory, command below
```
virtualenv env  
source ./env/bin/activate  
pip install -r requirements.txt  
```

How to use
---------------
1. Connect to the WiFi whose SSID is KYOTO_WIFI02
2. Perform the following
```
python auth.py
```
